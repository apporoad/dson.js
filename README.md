## 常用场景
1. 对json校验
2. format json
3. 不固定json结构的抽取和校验
4. 支持扩展，对非编程场景的二次封装使用
5. 支持嵌套、模板嵌套等，支持复杂的声明式数据查询、渲染、校验，类sql，可以动态拼接实现动态逻辑
6. 可以支持序列化，支持网络传播（待实现）
7. 声明式及模板方式实现逻辑，使逻辑可视化，提高可读性，提高可修改性
8. 解析excel
9. 解析html
